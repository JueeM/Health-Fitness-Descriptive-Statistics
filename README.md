# Health & Fitness Descriptive Statistics

Descriptive statistics analysis on a messy health dataset using Python, pandas, scipy, and seaborn.

## What it covers
- Data cleaning (missing values, junk strings, outliers, duplicates)
- Central tendency, spread, skewness, kurtosis
- Z-scores and outlier flagging
- Grouped analysis by activity level, gender, diet type, age band
- Correlation deep dive (1,275 column pairs ranked)
- 7 visualizations including heatmaps and scatter matrix

## Files
- `fitness_messy.csv` — dataset (515 rows, 54 columns)

## Libraries
pandas, numpy, scipy, matplotlib, seaborn
